# Blockchain DApp Lab

Laboratorio de desarrollo de aplicaciones descentralizadas (DApps) sobre Ethereum usando Hardhat.

## Stack

- **Hardhat** - Entorno de desarrollo y testing para Smart Contracts
- **Solidity** - Lenguaje de contratos inteligentes
- **Ethers.js** - Biblioteca para interactuar con la EVM
- **Chai** - Framework de testing
- **TypeChain** - Tipados autogenerados para contratos

## Comandos

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat ignition deploy ./ignition/modules/Lock.js
```
